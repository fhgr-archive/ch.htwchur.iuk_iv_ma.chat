Introduction

The following services can be used to implement the HTW Communicator. For data exchange JSON is used.

Service login.php

Description

The login-Service is used to register a user/password.

Parameter

user
The name of the user to be logged in.
password
The password of the user to be logged in.

Result

The result is either true or a descriptive error message.

Test

Service users.php

Description

The users-Service lists all registered users.

Parameter

no parameter

Result

The result is an array of user names.

Test

Service chat.php

Description

The chat-Service is used to send a text from an user to another.

Parameter

user1
The name of the user who sends the text.
user2
The name of the user who receives the text.
text
The text to send.

Result

The result is either true or a descriptive error message.

Test

Service chats.php

Description

The chats-Service lists all available chat entries exchanges between two users.

Parameter

user1
The name of one of the users.
user2
The name of the other user.

Result

The result is a list of chat entries. Example: 
 [
   { "time":"2012-05-01 13:42:17","user1":"333","user2":"Fritzli","text":"Hallo Fritzli"},
   { "time":"2012-05-01 13:42:45","user1":"Fritzli","user2":"333","text":"Hallo 333"},
 ]
